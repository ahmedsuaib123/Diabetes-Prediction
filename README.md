# Diabetes Risk Prediction

This project uses machine learning to predict whether a patient has diabetes based on medical data.

## Features
- Data preprocessing
- ML prediction model
- API for prediction

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy

## How to Run
1. Clone the repository
2. Install requirements
3. Run the prediction script

- Web interface
